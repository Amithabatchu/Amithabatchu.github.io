## How to publish on GitHub Pages
1. Create a repo named:
   - `Amithabatchu.github.io` (recommended) OR any repo name (e.g., `portfolio`)
2. Upload these files:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `README.md`
3. Go to **Settings → Pages**
4. Under **Build and deployment**
   - Source: **Deploy from a branch**
   - Branch: **main** / **root**
5. Your site will be available at:
   - `https://amithabatchu.github.io` (if repo name is username.github.io)
   - or `https://amithabatchu.github.io/<repo-name>/`

## Customize
- Update projects inside `index.html`
- Theme toggle is built-in (Dark/Light)
